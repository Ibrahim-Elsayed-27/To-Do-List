
# 📝 To-Do List

A simple and stylish To-Do List web app built using **HTML**, **CSS**, and **Vanilla JavaScript**.

## ✨ Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persist tasks in **localStorage**
- Responsive and modern UI
